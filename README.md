# Rizara Meats Platform

Rizara Meats is a digital platform for ethical, traceable halal meat in Kenya.  
It manages livestock, aggregation, processing, and connects farmers to buyers while keeping full traceability.

---

## Features

- Farmer and livestock management
- Aggregation batch tracking
- Processing batch management
- Traceability of all livestock and products
- Admin dashboard
- PostgreSQL database with Alembic migrations
- Flask backend with SQLAlchemy ORM
- Secure authentication (Flask-Login)

---

## Requirements

- Python 3.12+
- PostgreSQL
- Virtual environment (`venv`)
- Node.js & npm (if using frontend separately)
